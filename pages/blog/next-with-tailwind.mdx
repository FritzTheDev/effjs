### Setting Up Our Next.JS App
Let's get started by creating a new Next app. There are a couple good ways to do this, but we're going to stick with doing it from scratch.
This tutorial assumes you have a recent version of Yarn package manager installed. NPM will do the job just as well, you'll just have to translate a couple of the commands used.

```sh

yarn init -y
yarn add react react-dom next
```
